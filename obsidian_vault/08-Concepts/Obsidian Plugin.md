# Obsidian Plugin

## Definition
Obsidian vault integration plugin providing 5 skills for managing notes, canvases, databases, and web content extraction.

## Usage in Maya
Provides vault access capabilities:
- **obsidian:obsidian-cli** - Interact with vault (notes, tasks, properties, search)
- **obsidian:obsidian-markdown** - Create/edit Obsidian Flavored Markdown
- **obsidian:json-canvas** - Create/edit visual canvases (.canvas files)
- **obsidian:obsidian-bases** - Database-like views with filters/formulas
- **obsidian:defuddle** - Extract clean markdown from web pages

## Installation
```bash
# Add marketplace
/plugin marketplace add kepano/obsidian-skills

# Install plugin
/plugin install obsidian@obsidian-skills

# Reload to activate
/reload-plugins
```
