# Obsidian Daily Notes - Quick Reference

## 🚀 Shortcut Commands for Daily Reports

### **Open Today's Daily Note**
- **Windows/Linux:** `Ctrl+D`
- **macOS:** `Cmd+D`
- **Command Palette:** `Ctrl/Cmd+P` → Type "Daily" → Select "Open today's daily note"
- **Click:** Calendar icon in left sidebar (daily notes plugin)

### What This Shortcut Does:
- ✅ Creates `YYYY-MM-DD.md` in the vault root if it doesn't exist
- ✅ Applies the template from `Templates/Daily.md`
- ✅ Opens today's note instantly
- ✅ Switches to the note if it already exists

---

## 📝 Creating Daily Notes (Super Fast Method)

### Option 1: Keyboard Shortcut (Fastest)
```
Ctrl+D (or Cmd+D on Mac)
```
That's it! Opens/creates today's note with template applied.

### Option 2: Command Palette
```
1. Ctrl+P (or Cmd+P)
2. Type: "Daily"
3. Press Enter on "Open today's daily note"
```

### Option 3: Left Sidebar
- Click the calendar icon
- Today's note opens/created automatically

---

## 📌 Template Configuration (Already Set Up)

Your daily notes are now configured to:
- **Template:** `Templates/Daily.md`
- **Location:** Vault root (`obsidian_vault/`)
- **Format:** `YYYY-MM-DD.md`

**No manual copying needed!** The shortcut automatically applies the template.

## ⚡ Daily Workflow Example

```
Ctrl+D → Opens today's blank note
```

## 🔗 Linking to Other Notes (Quick Commands)

### Insert Link to Another Note
- Type: `[[` then start typing note name
- Example: `[[Phase Architecture]]` → creates clickable link

### See Backlinks (What's linked to current note)
- Open Backlinks pane: `Ctrl+Shift+B`
- Shows all notes that reference current note

### Open Graph View (Visual connections)
- `Ctrl+Shift+G`
- See how your daily notes connect to decisions, bugs, architecture

## 💡 Pro Tips

### Daily Note Best Practices:
1. **Use Ctrl+D as your "home base"** - Jump there anytime
2. **Link everything** - Use `[[Note Name]]` liberally
3. **Tag consistently** - Add tags like `#phase-8`, `#testing`, `#bug-fix`
4. **Review yesterday first** - Check previous day's "Next Steps" section

### Linking Made Easy:
- `[[07-Logs]]` → Links to Logs section
- `[[04-Decisions/TTS Priority Queue]]` → Links to specific decision
- `[[2026-03-20]]` → Links to previous day's note
- `[[#Next Steps]]` → Links to heading within current note

### Template Variables (If Needed):
The template uses `{{date}}` which Obsidian automatically replaces with today's date.
